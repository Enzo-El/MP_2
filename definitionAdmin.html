DEFINITION OF TERMS

IMPORTANT TERMS TO REMEMBER:

pending
  1. Case No. - this is the case number of the complaint which the admin must always specify when coordinating with the complainant and respondent.

  2. Complainant - this is the person who have filled out the blotter form and submitted the same on this site.

  3. Residence Address - this is the residence address of the complainant.

  4. Email Address - this is the email address of the complainant. The admin must always take note of this properly and correctly so that he/she can coordinate the status and data needed to resolve and further escalate the complaint.

  5. Contact No. - this is the contact number of the complainant. The admin must always take note of this properly and correctly so that he/she can coordinate the status and data needed to resolve and further escalate the complaint.

  6. Complaint - this is the nature/type of complaint that the complainant is seeking to resolved with.

  7. Description - this is the brief description of the complaint being raised on this site.

  8. Date of Incident - this is the actual date of the incident/complaint being reported thereby.

  9. Status - this is the current state of the complaint. There are five major classifications in this area: "Pending", "For Verification", "Under Investigation", "For Mediation", and "Resolved".

  10. Action Taken - similarly, this is the current state of the action on the complaint reported that will reflect the "Status" defined above. There are five major classifications in this area: "Pending", "For Verification", "Under Investigation", "For Mediation", and "Resolved".
